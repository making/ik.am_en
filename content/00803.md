LocalでLLMを試すのに[Ollama](https://ollama.com/)が人気です。Spring AIには[Ollama用のChat Client](https://docs.spring.io/spring-ai/reference/api/clients/ollama-chat.html)が用意されていますが、Ollamaには[OpenAI API互換API](https://ollama.com/blog/openai-compatibility)も用意されているので、OpenAIへの切り替えも想定して、[OpenAI用のChat Client](https://docs.spring.io/spring-ai/reference/api/chat/openai-chat.html)を使ってOllamaにアクセスしてみます。

> [!WARNING]
> Spring AIのOpenAI Clientは1.0.0-M1時点で、Ollama含む多くの互換APIプロバイダーが実装していないAPIを使用しています。
> [こちらのコミット](https://github.com/spring-projects/spring-ai/pull/863)で使用するAPIが修正されたため、本記事ではSpring AI 1.0.0-SNAPSHOTを使用しています。


Accessing the OpenAI API with curl.

```
curl -s http://localhost:11434/v1/chat/completions \
  -H "Content-Type: application/json" \
  -d '{
   "model": "gemma:2b",
   "messages": [
      {"role": "user", "content": "Give me a joke."}
   ]
 }' | jq .
```
